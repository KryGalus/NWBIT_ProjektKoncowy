# NWBIT_ProjektKoncowy
Projekt końcowy z przedmiotu Narzędzia w branży IT / Lab 6,7 i 8

## Opis

Data Converter jest narzędziem umożliwiającym konwersję danych pomiędzy formatami XML, JSON i YAML. Program wczytuje dane z pliku w jednym z tych formatów i zapisuje je do pliku w innym formacie, zgodnie z określonymi parametrami.

## Funkcjonalności

- Wczytuje dane z pliku w formacie XML, JSON lub YAML.
- Konwertuje dane do formatu XML, JSON lub YAML.
- Zapisuje dane do nowego pliku w wybranym formacie.
- Obsługuje błędy związane z niepoprawnymi danymi wejściowymi i nieprawidłowymi formatami plików.

## Wymagania

- Python 3.8 lub nowszy
- Biblioteki Python: `xmltodict`, `pyyaml`, `pyinstaller`

## Instalacja

Aby zainstalować wymagane biblioteki oraz narzędzie `pyinstaller`, uruchom poniższe komendy:
